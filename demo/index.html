Write base config

Reprograms the base station's keypad range and RF channel. Only effective while connected and not voting.

Keypad programming

Put a keypad in programming mode (vendor-specific button combo — consult your hardware manual), then use these to read or assign an ID. Most users never need this — IDs are factory-assigned.

—

Voting

Sends a "start voting session" command to the base station. The base then broadcasts the session parameters over RF, which wakes up the keypads and lets their buttons register.
